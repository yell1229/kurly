# React Kurly

목표: 리엑트로 컬리 UI 구현 및 기능들 적용하기

## 기술정보

클라이언트: ReactJS

서버: NodeJS(Express라이브러리 서버구축)

DB: MySQL

## 설치 라이브러리 
### client
```bash
npm i react-router-dom
npm i axios
npm i react-icons
npm i react-bootstrap
npm i swiper
npm i react-daum-postcode
```

### server
```bash
npm init --yes
npm i express --save
npm i mysql2
npm i cors
npm i nodemon --save-dev
npm i jsonwebtoken
```
